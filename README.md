# Projeto Calculadora

Este projeto faz parte do meu processo de aprendizado em **desenvolvimento web**.
A aplicação consiste em uma calculadora funcional construída utilizando tecnologias fundamentais da web.

##  Demonstração

A calculadora permite realizar operações matemáticas básicas através de uma interface simples e responsiva.

## 🛠 Tecnologias utilizadas

HTML – Estrutura da página
CSS – Estilização da interface
JavaScript – Lógica da calculadora

## 📚 O que aprendi com este projeto

* Manipulação do DOM
* Eventos de clique em botões
* Estrutura de funções em JavaScript
* Organização de arquivos em projetos web

## ▶️ Como executar

1. Baixe ou clone o repositório
2. Abra o arquivo **index.html** no navegador
